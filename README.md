# Unit Testing with Catch2

- Catch2 on github: https://github.com/catchorg/Catch2

- How to use catch2: https://github.com/catchorg/Catch2/blob/devel/examples/100-Fix-Section.cpp

## Compiler

You need a C++20 compiler. On Ubuntu 22.04 based systems use.

- gcc12: `sudo apt install g++-12`
- clang: `sudo apt install clang-15`
